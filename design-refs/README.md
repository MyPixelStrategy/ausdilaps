# Design references — drop Construktion X exports here

Free Figma has no Dev Mode MCP, so we match the template from **image exports** instead. I read PNG/JPG files here directly and reverse-engineer the design system (palette, type scale, spacing, components).

## How to export (works on free Figma)
1. Open the [Construktion X community file](https://www.figma.com/community/file/1364629887635378607/construktion-x-construction-figma-template) → **"Open in Figma" / Duplicate** (free → lands in your Drafts).
2. Select a frame → in the right panel, **Export** → **PNG, 2x** → Export.
3. Save the PNGs into this folder (`design-refs/`).

## What I most want (name them roughly so I can tell them apart)

- `services-*.png` — a services list page **and** a single service detail page

- anything else you personally like the look of

SVG exports of icons or the logo are welcome too. Once they're here, tell me and I'll build the AusDilaps design system from them.
